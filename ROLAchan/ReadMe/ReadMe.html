ROLAchanについて

使用に当たって

フォントをインストールして頂くと、ワープロ、グラフィックソフトなどの
アプリケーションでこのフォントを使って自由にデザイン、印刷することができます。

フォントの使用許諾

  1. フォントデータの複製、改変、再配布を禁ずる
  2. 予告なしにフォントのアップデートを行う場合があります
  3. 当フォントを使用した際に発生した機器およびソフトウエアの故障、
    誤動作、誤出力、ウイルスへの感染などに関して責任を負わないものとする

概要

ROLAchanは、幾何学図形を基調とした都会的なゴシックフォント。
アルファベット内の窮屈な余白を可能な限り取り除き、
文章全体で明るく元気な印象を持たせる書体です。
フリーフォントのため、可読性より上記コンセプトを優先しており
小文字の「S」がどうしても厳しい..という方に向けて
差し替えても自然に見えるフォントについて下記にまとめています。

https://ozawa.design/store/rolachan/

インストール

  1. フォントをダウンロードしたら圧縮されたファイルを解凍します。
    まず+Lhaca、StuffIt Expander、WinRAR、Lhaplus等の解凍ソフトを使用し
    解凍してフォントファイルにして下さい。
  2. フォントをパソコンにインストールします。
    お使いのパソコンのOSの標準的なインストール方法に従ってください。

ライセンス

ROLAchanにおいて、使用者がフォントライセンスを取得する必要はありません。
「フォントの使用許諾」をご理解の上ご利用ください。

サポートについて

ROLAchanはフリー配布のフォントのためサポート対象外です。

フォント商用利用・再配布・プログラム埋め込み・ウェブフォントについて

作成した印刷物およびデジタルコンテンツ・ソフトウェアにつき、その商用・非商用に関わらず放送、通信、各種記録メディアなどの媒体の形式も問わず、自由に使用をする事が可能です。
アプリ・ゲームに内蔵する場合、プログラムにエンベッド（埋め込み）をする場合、オンラインサーバー上に置く場合、ウェブフォントとしてサーバー上にアップロードする場合においても可能です。
フォントデータ自体を販売することのみ禁じます。
